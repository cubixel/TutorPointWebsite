File Directory

The CUBIXEL file directory contains all documentation that is associated with the CUBIXEL company, the TutorPoint product. The purpose of the CUBIXEL file directory is to assist the University of York Department of Electronic Engineering in the marking procedure for the Software Engineering Project Assessment.
